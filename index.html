Projects

Walking Tour Creator

The walking tour creator was a project I worked on with other members of a group as part of my universty degree. The project involved creating a web applicaition, using PHP and JavaScript, that displays walks a user had taken; recorded using an android application. I produced the web application to illustrate where the user had walked. Coordinates of the walk were recorded from a phone and uploaded to a database, where the web application then recieved the results. The coordinates were then placed on a a map using the Google Maps API to produce a route and replicate where the user walked.

File Upload Site

For my dissertation I produced a file upload site using the Laravel framework and VueJS for use by teachers and students. This application allows users to log in a upload their files onto the server which can then be sent to their teachers for marking. Teachers can also log in and see what files have been sent to them for marking as well as being able to see what their students have been uploading.
